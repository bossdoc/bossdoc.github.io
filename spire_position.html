Spire Position¶

Spire Position contains all the functions related to the analysis on the received sSAT AIS feed .

Functions¶

vast.management.commands.spire_position.check_geo_fencing(feed_list=None)¶

Used to check whether the vessel positions as per the latest AIS feed lie inside any of the Geo fencing polygon or not and also send email to the respective user.

Take input feedlist as dictionary with key as mmsi and value as dictionary containing AIS feed information

vast.management.commands.spire_position.check_seca_eca_feeds(vessels_feed)¶

this function return the two dictionaries: first one returns all the vessels inside ECA Second one returns all the vessels outside ECA

vast.management.commands.spire_position.create_geo_fencing_feed_list_db()¶

This function is just for testing purpose. Don’t want to call the API for testing purpose.

vast.management.commands.spire_position.get_geo_fence_alert_types(geo_fence_id=None)¶

We have created a set of alerts for a given geo fence (polygon) This function return all the geo fence alert type of a given geo_fence id If geo_fence_id is not passed it will return the alert type of all the active geo fence polygons return dictionary with key as geo_fence id and value as a list of dictionary of alert type

vast.management.commands.spire_position.get_plan_deviation_alert(vessel_list, alert, flag_type_for_each_vessel)¶

Function return if vessels follow the recommended route provided by the BOSS by cross verifying the position received as per the current AIS if not it will mark that vesel as “plan_deviatoin” as true and if everything is good it marked as False Input : List of vessels, flag_type_for_each_vessel is a dictionary with key as vessel and value as a dcitionary with speed_alert, plan_deviation, inside_polygon key. This is a return type void function, it only update the value of flag_type_for_each_vessel dictionary

vast.management.commands.spire_position.get_speed_value_alert(vessel_list, alert, flag_type_for_each_vessel)¶

Function return if vessels follow the speed range assgined by the user as per the current AIS if not it will mark that vesel as “speed_alert” true and if everything is good it marked as False Input : List of vessels, flag_type_for_each_vessel is a dictionary with key as vessel and value as a dcitionary with speed_alert, plan_deviation, inside_polygon key. This is a return type void function, it only update the value of flag_type_for_each_vessel dictionary

vast.management.commands.spire_position.send_alert_for_a_geo_fence(geo_fence_dict)¶

This function create an alert type dictionary that wheteher vessel follow speed inside the polygon or follow our recommendation as per the voyage plan and send email as well. Input : Dicitonary contains the basic information of geo_fence. this dictionary contains a key “inside_vessels” which is a list of all vessels inside the polygon
