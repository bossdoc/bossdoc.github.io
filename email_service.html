Parse body and add images to the email. Will be called implicitly from create_email_content() This will save the attached images - be in base64 or attached as img tag as an email attachment. send_email(subject, body, recipients, attachments=[], cc_list=None, bcc_list=None, message_uid=None, first_email=True)¶

Send an email using SMTP receiver_list: Takes string as well as list of string cc_list: Takes string as well as list of string attachments:

Parameters:
  • subject (str) – Email Subject
  • body (str) – Body
  • recipients (str or list(str)) – Email recipient (BCC and CC should be handled separately)
  • attachments (list(str), optional) – Takes list of file path for attachments, defaults to []
  • cc_list (str or list(str), optional) – CC Email list, defaults to None
  • bcc_list (str or list(str), optional) – BCC Email list, defaults to None
  • message_uid (str, optional) – Copilot Case-id (Any unique id to be used as email chain primary key), defaults to None
  • first_email (bool, optional) – Use this along with message_uid to distinguish b/w new chain or old chain - True implies this is the first email in this chain, defaults to True

read_email_chain(sender=None, email_uid=None, receiver=None, subject=None, historic_days=10, read_last_email=False, select_criteria='Inbox')¶

Read email chains methode for IMAP server Takes multiple filters for fetching emails

Parameters:
  • sender (str, optional) – Email Sender- In case you want to fetch from ‘Sent’, defaults to None
  • email_uid (int, optional) – Email uid from where you want to fetch emails - INCLUSIVE, defaults to None
  • receiver (str, optional) – In case you want to fetch on the basis of who received the email(To), defaults to None
  • subject (str, optional) – Subject for filtering, defaults to None
  • historic_days (int, optional) – Cap on days to be fetched, defaults to 10
  • read_last_email (bool, optional) – If you want to fail only the last sent email works well in Copilot Email Saving Mechanism, defaults to False
  • select_criteria (str, optional) – Email Folder - Sent, Inbox, defaults to ‘Inbox’
Returns:

List of email metadata dictionary

list(dict)
